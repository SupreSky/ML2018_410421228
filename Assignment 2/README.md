# Handwritten Character Recognition

本 project 利用 Stochastic Gradient Descent(SGD) 和 C-Support Vector Classification(SVC) 兩種機器學習方法和各種 Dimension reduction 方法，來解決以 MNIST 作為 dataset 的手寫字元辨識問題。

|   Method    | SGD           | SVC   |
| ----------- |:-------------:| -----:|
| auto        |       |   |
| full        |       |   |
| arpack      |       |   |
| randomized  |       |   |
